# Spotify-Display
it's basically a spotify display which would make it efficient and easy to control, change, pause or play the song, it contains an esp32 to connect to wifi and commit changes, a small screen to show things and three keyboard switches
i'm a beginner so my biggest inspiration to start this project is to learn something new and to be honest its kind of interesting too, im loving this, i'm still learning though, make sure to leavea a like!
